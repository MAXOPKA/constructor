# Constructor CMS

Content management system on Ruby on Rails

## Installation

### Add to Gemfile

    gem 'constructor-cms'    

### Install

    bundle install

### Migrate
    
    rake constructor_core:install:migrations
    rake constructor_pages:install:migrations
    
    rake db:migrate
